# Backend (NestJS + Prisma)

## Preferred way (from repo root)

```bash
pnpm install
pnpm dev:backend
```

Or run both apps together:

```bash
pnpm dev
```

## Local-only run (from this folder)

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

`prisma generate` runs automatically on install (`postinstall` script).
